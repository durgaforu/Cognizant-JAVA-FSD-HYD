1. Object

-> java.lang.Object is the super class to all the clases in java by default

-> This class contains methods which are common to any obejct in java



2. String
--------------
	Two types of objects
	1. mutable : content of the object can be changed/modified
		ex: StringBuffer
 	2. immutable : content of the object can not be changed / modified
		ex: String and all wraper classes
